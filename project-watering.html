Arrosage Automatique

Projet Personnel

Système automatisé pour la gestion de l'eau avec interface web.

Présentation du Projet

Le Défi

L'objectif de ce projet est de concevoir un système d'arrosage automatique intelligent, contrôlable via une interface web locale. Il intègre la gestion des pompes, des capteurs, et une planification horaire, le tout piloté par un microcontrôleur.

Architecture & Interface Web

Le système est basé sur un microcontrôleur programmé en C++ (framework Arduino/PlatformIO). Il héberge une page web interactive permettant de configurer les plages d'arrosage et déclencher manuellement les pompes.

Démonstration de l'interface Web en fonctionnement

Assemblage Matériel

Le contrôleur est intégré dans un coffret étanche. Les composants incluent des relais pour piloter les pompes et une alimentation dédiée.
